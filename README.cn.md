# Web UI Framework for AI
- 人类你好，这个框架是为Ai而设计的，
# AIGU - 为AI而设计的现代化 WEB UI 框架（AI Generator UI）
- 完整的现代化UI框架特性，依赖变更等
- 完整的 __WebComponts__ 支持 
- 超简洁直观的 __ComponentJs__ 语法
- AI 编辑支持
- 无 __"脚手架"__ 设计，无 __"VDOM"__ ，实时渲染
- 更新原有wcex实现UMD引入为标准ES import 和 importmap
- 其他特性继承 __"wcex"__ 项目
- ...

# 对 TypeScript说不
为什么要对 __TypeScript__ 说不？_typescript_ 本质就是一个 _javascript_ 的类型系统，这个系统是给 _IDE_ 和 _人_ 使用的，而 __AIGU__ 是为 __AI__ 而设计，因此没有必要使用 _typescript_，这只会增加更多不必要的问题。

# 增强 WebComponent 组件规范
- 完整符合可增强WebComponent规范。
- 将Js完整融入Dom，相比较传统的Js引入，多级作用域和快捷绑定，使Dom结构更加清晰和易于编辑，更适合于Ai进行处理。
- 详情参考: [增强 WebComponent 组件规范](./docs/component.md), 集成esm-dev

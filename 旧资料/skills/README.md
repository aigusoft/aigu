# WCEX 使用手册合集

这套手册基于仓库中的文档、示例与工具源码重新整理，目标是为自动化智能体与开发者提供一份覆盖框架特性的权威参考。所有章节均采用 Markdown，拆分为若干同级文件，方便检索、复用与模型训练。

## 目录速览
- [01-框架概览](01-框架概览.md) —— 设计理念、核心卖点、和 Vue 等框架的差异点。
- [02-快速开始与环境搭建](02-快速开始与环境搭建.md) —— CDN 方式与 CLI 工作流、必备元数据配置。
- [03-项目结构与组件命名](03-项目结构与组件命名.md) —— 标准目录规划、模板文件、标签命名转换规则。
- [04-组件结构与作用域](04-组件结构与作用域.md) —— `<template>`、属性声明、`meta scope`、脚本注入与本地作用域。
- [05-数据绑定语法详解](05-数据绑定语法详解.md) —— `$`/`:` 绑定、双向/单选/多选绑定、表达式求值规则。
- [06-事件与生命周期](06-事件与生命周期.md) —— 事件修饰符、内置生命周期钩子、`@timer`、监听模式。
- [07-条件循环与动态渲染](07-条件循环与动态渲染.md) —— `$if`、`$show`、`$for`、动态组件与插槽工具属性。
- [08-样式系统与主题](08-样式系统与主题.md) —— Shadow DOM 样式、动态样式字面量、语义配色与 `$Colors` 实用方法。
- [09-语义化配色](09-语义化配色.md) —— `$color` / `$Colors` 架构、Hsla 快捷语法、主题切换与持久化策略。
- [10-模块加载与脚本集成](10-模块加载与脚本集成.md) —— `meta module`、依赖自动安装、`<script scope>` 的 TS/第三方库用法。
- [11-路由系统](11-路由系统.md) —— Hash 分区路由格式、`$router` API、`route` 组件与导航协同。
- [12-内置工具与 Scope API](12-内置工具与 Scope API.md) —— `Scope` 工具方法、调试助手、异步辅助函数与插件扩展管线。
- [13-CLI 与开发流程](13-CLI 与开发流程.md) —— `wcex` 命令详解、热更新原理、构建与模块打包。
- [14-UI 组件库参考](14-UI 组件库参考.md) —— `@wcex/ui` 中常用组件概览与互操作要点。
- [15-测试与最佳实践](15-测试与最佳实践.md) —— UI 自动化测试、性能建议、组织大型项目的经验。
- [16-插件使用](16-插件使用.md) —— 插件生命周期、注册示例以及 `$color` / `$monitSize` 等内置能力。
- [17-开发调试](17-开发调试.md) —— Dev Panel、模板注释帮助、热更新与日志排查流程。
- [99-附录-速查表](99-附录-速查表.md) —— 常用指令、事件、颜色语法与 CLI 命令速览。

## 使用建议
- 按章节从基础到进阶循序阅读，可直接复制代码示例至 `wcex` Playground 或本地项目验证。
- 每份文档均面向自动化智能体撰写，尽量将语法、默认行为与异常情况显式列出，便于模型生成正确代码。
- 若后续仓库新增能力，请在对应章节补充；第三方模块或 UI 组件扩展可附加到 13、14 章节。

> 为保证一致性，请始终以本目录中的文件作为权威引用源。若需要英文或多语言版本，可在同级目录内创建 `*-en.md` 等对照文档。
